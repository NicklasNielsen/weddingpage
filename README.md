# Wedding Page Draft

A simple static wedding info page with sections for location, schedule, hotels, registry, and contact.

## Preview locally

Open `index.html` in your browser, or run a local server from the folder:

```bash
cd /home/nni/Desktop/source/weddingpage
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Next step

Once you like the page, you can add your real text and registry links, then deploy the folder as static files.
